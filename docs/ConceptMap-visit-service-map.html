Argonaut Scheduiing Visit Type to Service Type Mapping (http://fhir.org/guides/argonaut-scheduling/ConceptMap/visit-service-map)

Mapping from http://fhir.org/guides/argonaut-scheduling/ValueSet/visit-type to http://fhir.org/guides/argonaut-scheduling/ValueSet/service

DRAFT. Published on Oct 30, 2017 6:49:44 PM by Argonaut Project.

The source concepts are Argonaut Visit Types which are a prioritized valueset of the most common appointment types for scheduling and have been identified as the baseline use cases for the Argonaut scheduling Implementation Guide. The Visit types are mapped to the target concepts from the Argonaut Service Types Valueset in order to align with the FHIR Appointment, Schedule and Slot resources.
